# CodeAlpha Image Gallery

## 📌 Project Overview
This is a modern and responsive Image Gallery built using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## 🚀 Features
- Responsive image grid layout
- Click to view image in full-screen modal
- Next and Previous navigation buttons
- Keyboard navigation support
- Smooth animations and transitions
- Image filtering by categories (Nature, Animals, People)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📷 Preview
A clean and professional gallery UI with modern design and interactive features.

## 📎 How to Run
1. Download the project
2. Open `index.html` in your browser
